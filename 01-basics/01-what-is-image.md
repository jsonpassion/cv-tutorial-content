# 이미지란 무엇인가

## 한 줄 요약
컴퓨터에게 이미지는 **숫자로 가득 찬 엑셀 시트**입니다.

## 비유로 이해하기

엑셀을 떠올려보세요. 수천 개의 칸이 있고, 각 칸에 숫자가 들어있죠.

이미지도 똑같습니다:
- 각 칸 = **픽셀(pixel)**
- 각 숫자 = **밝기 값** (0~255)
- 0 = 완전히 검은색
- 255 = 완전히 흰색

```
예: 5x5 흑백 이미지

  0   0   0   0   0
  0  255 255 255  0
  0  255  0  255  0
  0  255 255 255  0
  0   0   0   0   0

→ 이게 바로 "O" 모양입니다!
```

## 기술적 설명

이미지는 **2D 또는 3D 배열(array)**로 표현됩니다.

| 이미지 유형 | 배열 형태 | 예시 |
|------------|----------|------|
| 흑백(Grayscale) | (높이, 너비) | (480, 640) |
| 컬러(RGB) | (높이, 너비, 채널) | (480, 640, 3) |

## 코드로 확인하기

```python
import numpy as np
from PIL import Image

# 이미지 파일 열기
img = Image.open("photo.jpg")

# NumPy 배열로 변환
arr = np.array(img)

# 배열의 형태 확인
print(f"이미지 크기: {arr.shape}")
# 출력 예: (480, 640, 3)
# → 480줄 x 640칸 x 3채널(RGB)

# 특정 픽셀의 값 확인 (왼쪽 위 모서리)
print(f"첫 번째 픽셀: {arr[0, 0]}")
# 출력 예: [142, 87, 63]
# → R=142, G=87, B=63 (갈색 계열)
```

## 핵심 포인트

1. **이미지 = 숫자 배열**: 컴퓨터는 "그림"을 보는 게 아니라 숫자를 처리합니다
2. **픽셀 값의 범위**: 보통 0~255 (8비트)
3. **채널**: 흑백은 1채널, 컬러는 3채널(RGB)
